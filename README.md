## 🔷 PREREQUISITES
For this tutorial and to use OPGIT, we assume that you usually use Git for your GitHub projects, versioning and forks, and that you have a specific working directory.  

💡 You can verify if Git is installed on your system, by typing the foloowing command in a terminal :
```code
git --version
```
  
If you use GitHub to share your work or participate in a repository, we also assume you have a GitHub user account.
